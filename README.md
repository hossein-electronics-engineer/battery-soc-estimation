# Battery SOC Estimation using Equivalent Circuit Models

## Objective
This project explores model-based State of Charge (SOC) estimation for lithium-ion batteries.

## Approach
- Equivalent circuit battery model (RC model)
- Simulation of voltage and current behavior
- SOC estimation methods (to be implemented)

## Project Structure
- src: source code
- data: input data
- results: outputs
- figures: plots

## Status
Work in progress – initial modeling phase
